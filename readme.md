## 关于

一个简单的反向链接插件，能用但有待进一步完善。

## 功能

* 带有标题、创建时间、和 tag 信息的反向链接。
* 默认开启的预览。
* 默认关闭的全局发现（全局搜索）。

## 安装

clone 到 wiki 目录的 plugins 子目录下即可。
